## TRY FLOWISE


**HOW TO INSTALL FLOWISEAI**

```bash
# go to dir
cd /Users/brunoflaven/Documents/01_work/blog_articles/ia_using_flowiseai/
# create dir
mkdir flowiseai
# go to the dir flowiseai
cd flowiseai
# install flowise
npm install flowise

# or install flowise globally
npm install -g flowise
npm uninstall -g flowise

# start flowise
npx flowise start

# check the site
http://localhost:3000/
```
