# Programming Languages for 3D Software

## 1. Programming Language Support

### **Rhinoceros 3D**
- **Python**: ‚úÖ Full support via RhinoScriptSyntax and rhinoscriptsyntax library
- **JavaScript**: ‚ùå Not natively supported
- **Additional**: Also supports C# and VB.NET through RhinoCommon API
- **Use cases**: Automation, parametric modeling, custom tools, grasshopper scripting

### **AutoCAD**
- **Python**: ‚ö†Ô∏è Limited/indirect support (requires pyautocad library or COM interface)
- **JavaScript**: ‚ùå Not natively supported in desktop AutoCAD
- **Primary languages**: AutoLISP, Visual LISP, VBA, .NET (C#, VB.NET)
- **Note**: AutoCAD's web version supports JavaScript for customization

### **3ds Max**
- **Python**: ‚úÖ Full support via MaxPlus API and pymxs
- **JavaScript**: ‚ùå Not natively supported
- **Primary language**: MAXScript (proprietary scripting language)
- **Use cases**: Automation, custom tools, rendering pipelines, batch processing

# Rhino Surface Area Analysis Scripts

## 1. Using RhinoScriptSyntax (RhinoPython)

```python
import rhinoscriptsyntax as rs
import scriptcontext as sc

def analyze_and_color_surfaces():
    """
    Selects all surfaces, analyzes their areas, and color-codes them based on size ranges
    """
    
    # Select all surface objects
    surfaces = rs.GetObjects("Select surfaces to analyze", rs.filter.surface, preselect=True)
    
    # If no selection, get all surfaces in the document
    if not surfaces:
        surfaces = rs.ObjectsByType(rs.filter.surface)
    
    if not surfaces:
        print("No surfaces found in the document.")
        return
    
    # Calculate areas for all surfaces
    areas = []
    for surf in surfaces:
        area = rs.SurfaceArea(surf)
        if area:
            areas.append(area[0])  # area[0] is the total area
        else:
            areas.append(0)
    
    # Find min and max areas
    min_area = min(areas)
    max_area = max(areas)
    area_range = max_area - min_area
    
    print("\n=== Surface Area Analysis ===")
    print("Total surfaces: {}".format(len(surfaces)))
    print("Minimum area: {:.2f}".format(min_area))
    print("Maximum area: {:.2f}".format(max_area))
    print("Area range: {:.2f}\n".format(area_range))
    
    # Define color ranges (5 categories)
    colors = [
        (0, 255, 0),      # Green - smallest
        (173, 255, 47),   # Yellow-Green
        (255, 255, 0),    # Yellow - medium
        (255, 165, 0),    # Orange
        (255, 0, 0)       # Red - largest
    ]
    
    # Color-code each surface based on its area
    for i, surf in enumerate(surfaces):
        area = areas[i]
        
        # Calculate which color category (0-4)
        if area_range > 0:
            normalized = (area - min_area) / area_range
            color_index = int(normalized * (len(colors) - 1))
            color_index = min(color_index, len(colors) - 1)  # Ensure within bounds
        else:
            color_index = 0  # All same area
        
        # Apply color to the surface
        rs.ObjectColor(surf, colors[color_index])
        
        # Print individual surface info
        print("Surface {}: Area = {:.2f}, Color = {}".format(
            i + 1, area, ["Small", "Small-Medium", "Medium", "Medium-Large", "Large"][color_index]
        ))
    
    print("\n=== Color Legend ===")
    print("Green: Smallest areas")
    print("Yellow-Green: Small-medium areas")
    print("Yellow: Medium areas")
    print("Orange: Medium-large areas")
    print("Red: Largest areas")
    
    rs.Redraw()

# Run the function
if __name__ == "__main__":
    analyze_and_color_surfaces()
```

## 2. Using Pure Python with RhinoCommon

```python
import Rhino
import scriptcontext as sc
import System.Drawing.Color as Color

def analyze_and_color_surfaces_rhinocommon():
    """
    Uses RhinoCommon API to analyze and color-code surfaces by area
    """
    
    doc = sc.doc
    
    # Get all surface objects from the document
    surface_objects = []
    for obj in doc.Objects:
        if obj.ObjectType == Rhino.DocObjects.ObjectType.Surface or \
           obj.ObjectType == Rhino.DocObjects.ObjectType.Brep:
            surface_objects.append(obj)
    
    if not surface_objects:
        print("No surfaces found in the document.")
        return
    
    # Calculate areas for all surfaces
    areas = []
    for obj in surface_objects:
        geom = obj.Geometry
        
        # Get area based on geometry type
        if isinstance(geom, Rhino.Geometry.Brep):
            mass_props = Rhino.Geometry.AreaMassProperties.Compute(geom)
            if mass_props:
                areas.append(mass_props.Area)
            else:
                areas.append(0)
        elif isinstance(geom, Rhino.Geometry.Surface):
            mass_props = Rhino.Geometry.AreaMassProperties.Compute(geom)
            if mass_props:
                areas.append(mass_props.Area)
            else:
                areas.append(0)
        else:
            areas.append(0)
    
    # Find min and max areas
    min_area = min(areas)
    max_area = max(areas)
    area_range = max_area - min_area
    
    print("\n=== Surface Area Analysis (RhinoCommon) ===")
    print("Total surfaces: {}".format(len(surface_objects)))
    print("Minimum area: {:.2f}".format(min_area))
    print("Maximum area: {:.2f}".format(max_area))
    print("Area range: {:.2f}\n".format(area_range))
    
    # Define color ranges using System.Drawing.Color
    colors = [
        Color.FromArgb(0, 255, 0),        # Green
        Color.FromArgb(173, 255, 47),     # Yellow-Green
        Color.FromArgb(255, 255, 0),      # Yellow
        Color.FromArgb(255, 165, 0),      # Orange
        Color.FromArgb(255, 0, 0)         # Red
    ]
    
    category_names = ["Small", "Small-Medium", "Medium", "Medium-Large", "Large"]
    
    # Color-code each surface
    for i, obj in enumerate(surface_objects):
        area = areas[i]
        
        # Calculate color category
        if area_range > 0:
            normalized = (area - min_area) / area_range
            color_index = int(normalized * (len(colors) - 1))
            color_index = min(color_index, len(colors) - 1)
        else:
            color_index = 0
        
        # Create object attributes with new color
        attributes = obj.Attributes
        attributes.ObjectColor = colors[color_index]
        attributes.ColorSource = Rhino.DocObjects.ObjectColorSource.ColorFromObject
        
        # Modify the object
        doc.Objects.ModifyAttributes(obj, attributes, True)
        
        # Print info
        print("Surface {}: Area = {:.2f}, Category = {}".format(
            i + 1, area, category_names[color_index]
        ))
    
    print("\n=== Color Legend ===")
    print("Green: Smallest areas")
    print("Yellow-Green: Small-medium areas")
    print("Yellow: Medium areas")
    print("Orange: Medium-large areas")
    print("Red: Largest areas")
    
    doc.Views.Redraw()

# Run the function
if __name__ == "__main__":
    analyze_and_color_surfaces_rhinocommon()
```

## Key Differences Between the Two Approaches:

### **RhinoScriptSyntax (Script 1)**
- ‚úÖ **Easier to learn** - simpler, more intuitive syntax
- ‚úÖ **Less code** - wrapper functions handle complexity
- ‚úÖ **Good for beginners** - closer to RhinoScript syntax
- ‚ö†Ô∏è **Slower performance** - additional wrapper layer
- üìù Uses functions like `rs.SurfaceArea()`, `rs.ObjectColor()`

### **RhinoCommon (Script 2)**
- ‚úÖ **Faster performance** - direct API access
- ‚úÖ **More control** - access to full Rhino object model
- ‚úÖ **Better for complex tasks** - professional development
- ‚ö†Ô∏è **Steeper learning curve** - more verbose code
- üìù Uses classes like `Rhino.Geometry.AreaMassProperties`
